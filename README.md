## 🕹️ How to Play

- The computer randomly places 5 ships on a 10x10 grid.
- Your goal: sink them all by guessing their coordinates.
- Symbols:
  - `X` = hit
  - `O` = miss
  - Blue cells = unexplored

## 🛠️ How to Run

### On macOS/Linux:
```bash
g++ SeaBattle.cpp -o SeaBattle
./SeaBattle
